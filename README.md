A difficulty-based maze generator and solver web app built using
**HTML5 Canvas, Vanilla JavaScript, and DFS maze generation**.

## Features
- Difficulty levels (Easy / Medium / Hard)
- Algorithmic maze generation (DFS)
- Real-time mouse path tracing
- Wall collision detection
- Time-based scoring system
- Local high score storage
- Replay support
- GitHub Pages deployment

## How to Play
1. Select a difficulty
2. Start from the green node
3. Trace the path without touching walls
4. Reach the red node to finish
